# HTTP与HTTPs区别

HTTP协议是超文本传输协议，是应用应用层，其中存在几个问题：
1. 请求信息是明文传输，容易被窃取
2. 没有身份验证，存在冒充风险
3. 数据完整性未校验，容易被盘算
HTTPS就是便可以解决以上问题，HTTPS在HTTP基础上加入了SSL/TLS，相当于在HTTP基础上加入了安全保护

#### 主要的区别

|  安全性   | HTTP  | HTTPS  |
|  ----  | ----  | ----  |
| 安全性  | 不安全 | 安全  |
| 默认端口  | 80 | 443  |
| 资源消耗  | 较少 | 较多  |
| 是否需要证书  | 不需要 | 需要  |
| 报文是否加密  | 明文 | 密文  |
