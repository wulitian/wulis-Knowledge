# POST与GET请求

|  请求方式   | get  | post  |
|  ----  | ----  | ----  |
| 数据包  | 产生一个tcp数据包 | 可能产生两个tcp数据包  |
| 编码方式  | 只进行url编码 | 支持多种编码  |
| 请求参数  | 参数在url中用&号链接 | 通过request body 传递参数  |
| 请求缓存  | 主动缓存 | 不会缓存除非设置缓存  |
| 收藏为书签  | 支持 | 不支持  |
| 历史记录  | 参数会完整的保留在浏览器历史记录 | 不会报错在历史记录中  |
| 参数数据类型  | ASCII | 没有限制  |
